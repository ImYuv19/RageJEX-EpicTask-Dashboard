# 🚀 RageJEX EpicTask Dashboard

A modern project and task management dashboard built with **React.js**, **Vite**, and **Tailwind CSS**. The system helps teams manage projects, track tasks, visualize dependencies, balance workloads, and monitor project progress through an intuitive interface.

## 🧠 Algorithms Used

### Critical Path Analysis

Uses DFS + Memoization to calculate the project's critical path and estimated completion duration.

### Dependency Cycle Detection

Prevents circular task dependencies using DFS-based graph traversal.

### Workload Balancing

Automatically suggests the least-burdened team member for task assignments.

### Undo / Redo System

Dual-stack state management for timeline history tracking.

### FIFO Notification Queue

Processes alerts in First-In-First-Out order.
